# Product Backlog

This document serves as our future product roadmap. It lists the high-level features and requirements slated for subsequent development phases.

## High-Level Requirements

### 1. Inbound Calling System
Allow the agent to accept incoming calls seamlessly via an inbound trunk configuration instead of just making outbound calls.

### 2. Automated Reminder Calls
Logic to automatically trigger outbound reminder sessions for patients based on specific trigger conditions or statuses.

### 3. WhatsApp & SMS Confirmation Automation
Integration with a webhook or external automation pipeline to send confirmation messages immediately after a call successfully disconnects.

### 4. Analytics & Lead Dashboard
A clean user interface to display call logs, summaries, status updates, and tracking data for the clinic owner.
